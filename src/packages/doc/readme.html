Documentation for Opt-Packages

The documentations for the individual packages can be found in the directory doc/english/pdf in PDF format and in the directory doc/english/html in HTML format. The names of the PDF files are simply <Name of Opt>.pdf; the HTML version can be found under <Name of Opt>/index.html.
